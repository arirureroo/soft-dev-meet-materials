## Challenge 1 — Add Course Form (POST)

- Implement the async submit handler in [script.js](script.js).
- Send a `fetch()` POST to the MockAPI endpoint, disable the submit button while waiting, then render the new course card only on success.
- On page load, fetch the existing courses and render them into the list.
- Use the existing markup in [index.html](index.html) and styles in [style.css](style.css).

## Challenge 2 — Quote Fetcher (GET)

- Implement the click handler in [script.js](script.js).
- Disable the button and show a loading state, then `fetch()` a random quote and render the quote + author.
- Handle errors with a fallback message and always re-enable the button.

Open [index.html](index.html) in a browser to test your work.
