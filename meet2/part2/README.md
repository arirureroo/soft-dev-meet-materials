## Meet 2 - Part 2

Two sub-topics:

1. Asynchronous JavaScript
2. Fetch API

### Asynchronous JavaScript (learning flow)

Read the files in this order:

1. [sync.js](sync.js) - synchronous flow
2. [callback.js](callback.js) - callback basics
3. [callback_hell.js](callback_hell.js) - nested callbacks
4. [promise.js](promise.js) - Promise chaining
5. [async.js](async.js) - async/await and parallel execution

### Fetch API

See [get.js](get.js) and [post.js](post.js) for simple GET and POST examples.

---

Run any file with Node.js, e.g.:

```bash
node get.js
```
